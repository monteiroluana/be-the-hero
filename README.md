# OmniStack Week 11.0 
**Be The Hero**: Application developed in the Ominstack 11.0 Week by Rocketseat.

## Technologies 
NodeJS, ReactJS, React Native

## Getting start

### Backend
```
$ cd backend

$ npm install

$ npx knex migrate:latest 

$ npm start
```

### Frontend
```
$ cd frontend

$ npm install

$ npm start
```

### Mobile
```
$ cd mobile

$ npm install

$ expo start
```
